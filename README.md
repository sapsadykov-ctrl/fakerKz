# fakerKz

Локализация для библиотеки Faker на Python для генерации реалистичных тестовых (фейковых) данных для Казахстана.

## Структура
Проект реализован на основе официальной документации Faker: https://faker.readthedocs.io/en/master/index.html 
и кода: https://github.com/joke2k/faker 

## Реализовано / Планы (Roadmap)
- [ ] Генерация индивидуальных идентификационных номеров (ИИН) с валидной контрольной суммой.
- [ ] БИН (Бизнес-идентификационный номер) для компаний.
- [ ] Казахские мужские и женские имена, отчества и фамилии.
- [ ] Шаблоны локальных номеров телефонов (Kcell, Beeline, Tele2, Altel).
- [ ] Автомобильные номера (старого и нового образца с регионами).
- [ ] Генерация адресов (города Казахстана, улицы).
